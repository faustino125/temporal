### Function `get_atp_investment` {#data_engineering.data_integration.src.mc_investment.mc_investment_flow.get_atp_investment}




>     def get_atp_investment(
>         df: pyspark.sql.dataframe.DataFrame
>     ) ‑> pyspark.sql.dataframe.DataFrame


This function performs two transformations:
1. Adds a new column atp_investment_flag:
   - "S" if inv_saldo_quetzalizado_val > 200000
   - "N" otherwise
2. Adds a new column producto_desc:
   - "pdi" if inv_id_producto == 5
   - "pfu" if inv_id_producto == 11
   - "pf"  if inv_id_producto == 13


Args
-----=
**```df```** :&ensp;<code>DataFrame</code>
:   Input Spark DataFrame containing at least the following columns:
                - inv_saldo_quetzalizado_val (dataframe): Source dataframe
                - inv_id_producto (dataframe): Source dataframe



Returns
-----=
<code>DataFrame</code>
:   Transformed DataFrame with two additional columns:
           - atp_investment_flag (string)
           - producto_desc (string)




### Function `mc_investment_flow` {#data_engineering.data_integration.src.mc_investment.mc_investment_flow.mc_investment_flow}




>     def mc_investment_flow()


Load, integrate  and saves result data in data lake.

The flow performs the following operations:
1. Loads data using the specified date range.
2. Processes and integrate features data.
3. Saves the processed data to the appropriate environment using the
   specified overwrite strategy.


Returns
-----=
<code>None</code>
:   This flow does not return a value, but saves the processed


features data.
