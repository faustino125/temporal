Builds customer universe and product-level aggregations.

Aggregates raw transformed data from each product into features grouped
by client and observation date.


Args
-----=
**```p_transformed_current```** :&ensp;<code>DataFrame</code>
:   Transformed current account data.


**```p_transformed_saving```** :&ensp;<code>DataFrame</code>
:   Transformed saving data.


**```p_transformed_customer```** :&ensp;<code>DataFrame</code>
:   Transformed customer data.


**```p_transformed_credit_card```** :&ensp;<code>DataFrame</code>
:   Transformed credit card data.


**```p_transformed_loan```** :&ensp;<code>DataFrame</code>
:   Transformed loan data.



Returns
-----=
<code>dict\[str, DataFrame]</code>
:   Dictionary with the following keys:
    - df_customer: Distinct customer universe.
    - df_customer_balance: Customer monthly income.
    - df_savings: Aggregated savings balance.
    - df_current: Aggregated current account balance.
    - df_loan: Aggregated loan installment.
    - df_credit_card: Aggregated credit card balances.




### Function `mc_cliente_flow` {#data_engineering.data_integration.src.mc_cliente.mc_cliente_flow.mc_cliente_flow}




>     def mc_cliente_flow()


Load, integrate  and saves result data in data lake.

The flow performs the following operations:
1. Loads data using the specified date range.
2. Processes and integrate features data.
3. Saves the processed data to the appropriate environment using the
   specified overwrite strategy.


Returns
-----=
<code>None</code>
:   This flow does not return a value, but saves the processed


features data.
