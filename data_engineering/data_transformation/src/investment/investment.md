### Function `investment_flow` {#data_engineering.data_transformation.src.investment.investment_flow.investment_flow}




>     def investment_flow()


Loads, transforms and saves investment features in the data lake.

The flow performs the following operations:
1. Loads clean investment data using the specified date range.
2. Transforms and processes the investment features data.
3. Saves the processed data to the appropriate environment using the
   specified overwrite strategy.


Returns
-----=
<code>None</code>
:   This flow does not return a value, but saves the processed


investment features data.
